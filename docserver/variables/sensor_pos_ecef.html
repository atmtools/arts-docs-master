Workspace Variable sensor_pos_ecef

As sensor_pos but using ECEF coordinates.

The sensor position is here specified as earth-centered, earth-fixed
(ECEF) coordinates (using standard definition of ECEF).

Probably only useful for 3D.

Column 1, 2 and 3 shall hold x, y and z coordinate, respectively.

Usage: Set by the user.

Unit:  [ m, m, m ]

Size:  [ number of measurement blocks, atmosphere_dim ]

Group: Matrix
