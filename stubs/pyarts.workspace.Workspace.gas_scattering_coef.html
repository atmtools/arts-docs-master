gas_scattering_coef

property Workspace.gas_scattering_coef

Spectrum of scattering coefficient matrices.

This variable contains the elements of the extinction matrix solely due to scattering.

Usage: Output of gas_scattering_agenda.

Units: [ m^-1. ]

Size: [fgrid, stokes_dim, stokes_dim]

Workspace methods that can generate gas_scattering_coef

  • gas_scattering_coefAirSimple()

  • gas_scattering_coefXsecConst()

Workspace agendas that can generate gas_scattering_coef

  • gas_scattering_agenda

Generic workspace methods that can generate or use gas_scattering_coef

See PropagationMatrix and/or Any

type:

WorkspaceVariable - holds type PropagationMatrix
