covmat_inv_block

property Workspace.covmat_inv_block

Holds matrices used to set the inverse blocks in covmat_sx and covmat_se.

The matrix contained in this block will be used as the inverse of the matrix contained in covmat_block.

Usage: Used by the retrievalAdd functions.

Workspace methods that can generate covmat_inv_block

  • retrievalDefInit()

Workspace methods that require covmat_inv_block

  • retrievalAddAbsSpecies()

  • retrievalAddCatalogParameters()

  • retrievalAddFreqShift()

  • retrievalAddFreqStretch()

  • retrievalAddMagField()

  • retrievalAddPointingZa()

  • retrievalAddPolyfit()

  • retrievalAddScatSpecies()

  • retrievalAddSinefit()

  • retrievalAddSpecialSpecies()

  • retrievalAddTemperature()
