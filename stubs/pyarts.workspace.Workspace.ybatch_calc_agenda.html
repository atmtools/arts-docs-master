ybatch_calc_agenda

property Workspace.ybatch_calc_agenda

Calculations to perform for each batch case.

Must produce a new spectrum vector (y) and Jacobi matrix (jacobian). See further ybatchCalc().

Parameters:

  • y (Vector) – The measurement vector. See y [OUT]

  • y_aux (ArrayOfVector) – Data auxilary to y. See y_aux [OUT]

  • jacobian (Matrix) – The Jacobian matrix. See jacobian [OUT]

  • ybatch_index (Index) – Index of batch case. See ybatch_index [IN]

Workspace methods that can generate ybatch_calc_agenda

  • ybatch_calc_agendaSet()

Workspace methods that require ybatch_calc_agenda

  • ybatchCalc()

Generic workspace methods that can generate or use ybatch_calc_agenda

See Agenda and/or Any

type:

WorkspaceVariable - holds type Agenda
