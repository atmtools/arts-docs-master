g0

property Workspace.g0

Gravity at zero altitude.

This variable is “little g” at the reference ellipsiod. That is, for Earth this is a value around 9.81 m/s2

Workspace methods that can generate g0

  • g0Earth()

  • g0Io()

  • g0Jupiter()

  • g0Mars()

  • g0Venus()

Workspace agendas that can generate g0

  • g0_agenda

Generic workspace methods that can generate or use g0
