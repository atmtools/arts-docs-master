get_n2_standard

pyarts.arts.predef.get_n2_standard(f_grid: pyarts.arts.Vector, rtp_pressure: float, rtp_temperature: float, x_n2: float) → pyarts.arts.Vector

Computes nitrogen absorption using Rosenkranz standard

Parameters:

  • f_grid (Vector) – Frequency grid [Hz]

  • rtp_pressure (float) – Pressure value [Pa]

  • rtp_temperature (float) – Temperature value [K]

  • x_n2 (float) – Ratio of nitrogen in the atmosphere in the range [0, 1]

Returns:

abs_coef – Absorption coefficients

Return type:

Vector
