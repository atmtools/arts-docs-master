Create a radiance profile operator

This is an experimental solution.

Author(s): Richard Larsson

Parameters:

  • spectral_radiance_profile_operator (SpectralRadianceProfileOperator, optional) ‚Äì An operator to create a spectral radiance profile. See spectral_radiance_profile_operator, defaults to self.spectral_radiance_profile_operator [OUT]

  • z_field (Tensor3, optional) ‚Äì The field of geometrical altitudes. See z_field, defaults to self.z_field [IN]

  • ppath_lmax (Numeric, optional) ‚Äì Maximum length between points describing propagation paths. See ppath_lmax, defaults to self.ppath_lmax [IN]

  • atmosphere_dim (Index, optional) ‚Äì The atmospheric dimensionality (1-3). See atmosphere_dim, defaults to self.atmosphere_dim [IN]

  • p_grid (Vector, optional) ‚Äì The pressure grid. See p_grid, defaults to self.p_grid [IN]

  • t_field (Tensor3, optional) ‚Äì The field of atmospheric temperatures. See t_field, defaults to self.t_field [IN]

  • nlte_field (EnergyLevelMap, optional) ‚Äì The field of NLTE temperatures and/or ratios. See nlte_field, defaults to self.nlte_field [IN]

  • vmr_field (Tensor4, optional) ‚Äì VMR field. See vmr_field, defaults to self.vmr_field [IN]

  • wind_u_field (Tensor3, optional) ‚Äì Zonal component of the wind field. See wind_u_field, defaults to self.wind_u_field [IN]

  • wind_v_field (Tensor3, optional) ‚Äì Meridional component of the magnetic field. See wind_v_field, defaults to self.wind_v_field [IN]

  • wind_w_field (Tensor3, optional) ‚Äì Vertical wind component field. See wind_w_field, defaults to self.wind_w_field [IN]

  • mag_u_field (Tensor3, optional) ‚Äì Zonal component of the magnetic field. See mag_u_field, defaults to self.mag_u_field [IN]

  • mag_v_field (Tensor3, optional) ‚Äì Meridional component of the magnetic field. See mag_v_field, defaults to self.mag_v_field [IN]

  • mag_w_field (Tensor3, optional) ‚Äì Vertical component of the magnetic field. See mag_w_field, defaults to self.mag_w_field [IN]

  • abs_species (ArrayOfArrayOfSpeciesTag, optional) ‚Äì Tag groups for gas absorption. See abs_species, defaults to self.abs_species [IN]

  • predefined_model_data (PredefinedModelData, optional) ‚Äì This contains predefined model data. See predefined_model_data, defaults to self.predefined_model_data [IN]

  • abs_cia_data (ArrayOfCIARecord, optional) ‚Äì HITRAN Collision Induced Absorption (CIA) Data. See abs_cia_data, defaults to self.abs_cia_data [IN]

  • xsec_fit_data (ArrayOfXsecRecord, optional) ‚Äì Fitting model coefficients for cross section species. See xsec_fit_data, defaults to self.xsec_fit_data [IN]

  • isotopologue_ratios (SpeciesIsotopologueRatios, optional) ‚Äì Contains the isotopologue ratios. See isotopologue_ratios, defaults to self.isotopologue_ratios [IN]

  • abs_lines_per_species (ArrayOfArrayOfAbsorptionLines, optional) ‚Äì A list of spectral line data for each tag. See abs_lines_per_species, defaults to self.abs_lines_per_species [IN]

  • T_extrapolfac (Numeric, optional) ‚Äì Temperature extrapolation factor (relative to grid spacing). Defaults to 0.5 [IN]

  • ignore_errors (Index, optional) ‚Äì Set to 1 to suppress runtime errors (and return NAN values instead). Defaults to 0 [IN]

  • verbosity (Verbosity) ‚Äì ARTS verbosity. See verbosity, defaults to self.verbosity [IN]
