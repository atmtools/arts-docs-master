gas_scattering_agenda

property Workspace.gas_scattering_agenda

Calculation of the gas scattering extinction and phase matrix.

This agenda calculates the gas scattering cross section and the normalized phase matrix for a specific incoming ( gas_scattering_los_in ) and outgoing (gas_scattering_los_out) direction. The scattering cross section is calculated along a propagtion path given by the propagation path variables rtp_pressure, rtp_temperature, and rtp_vmr.If gas_scattering_los_in and gas_scattering_los_out are empty vectors, then gas_scattering_mat is set empty. If gas_scattering_los_in and gas_scattering_los_out are not empty, then the phase matrix is calculated for the define incoming and outgoing direction.

Parameters:

  • gas_scattering_coef (PropagationMatrix) – Spectrum of scattering coefficient matrices. See gas_scattering_coef [OUT]

  • gas_scattering_mat (TransmissionMatrix) – Spectrum of normalized phase matrices. See gas_scattering_mat [OUT]

  • gas_scattering_fct_legendre (Vector) – Normalized phase function as Legendre series. See gas_scattering_fct_legendre [OUT]

  • f_grid (Vector) – The frequency grid for monochromatic pencil beam calculations. See f_grid [IN]

  • rtp_pressure (Numeric) – Pressure at a radiative transfer point. See rtp_pressure [IN]

  • rtp_temperature (Numeric) – Temperature at a radiative transfer point. See rtp_temperature [IN]

  • rtp_vmr (Vector) – Absorption species abundances for radiative transfer calculations. See rtp_vmr [IN]

  • gas_scattering_los_in (Vector) – Incoming line-of-sight for gas scattering. See gas_scattering_los_in [IN]

  • gas_scattering_los_out (Vector) – Outgoing line-of-sight for gas scattering. See gas_scattering_los_out [IN]

  • gas_scattering_output_type (Index) – Flag to select the output of the gas_scattering_agenda. See gas_scattering_output_type [IN]
