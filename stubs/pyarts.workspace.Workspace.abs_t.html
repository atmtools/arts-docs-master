abs_t

property Workspace.abs_t

List of temperatures to be used for the calculation of absorption coefficients.

In contrast to the global t_field, this is just a vector. Any absorption method should check that the length of this vector is the same as that of abs_p

Dimension: [p_grid]

Unit: K

Workspace methods that can generate abs_t

  • AbsInputFromAtmFields()

  • abs_lookupSetup()

  • abs_lookupSetupBatch()

  • abs_lookupSetupWide()

Workspace methods that require abs_t

  • abs_lookupCalc()

Generic workspace methods that can generate or use abs_t

See Vector and/or Any

type:
