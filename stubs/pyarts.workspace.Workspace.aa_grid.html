aa_grid

property Workspace.aa_grid

Azimuthal angle grid.

The azimutal angle grid, on which the cloudbox_field is stored. This grid is used for RT calculations inside the cloudbox, therefore one has to define it if the cloudbox is activated by the flag cloudbox_on. Furthermore the zenith angle grid is also usedfor RT calculations of clear-sky spectral_radiance_field. The grid must be sorted in increasing order, with no repetitions.

Usage: Set by the user.

Unit: degrees
