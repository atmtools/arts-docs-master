spectral_radiance_fieldFromOperatorPath

Workspace.spectral_radiance_fieldFromOperatorPath(self: pyarts.arts._Workspace, spectral_radiance_field: pyarts.arts.StokvecGriddedField6 | None = None, spectral_radiance_operator: pyarts.arts.SpectralRadianceOperator | None = self.spectral_radiance_operator, propagation_path_observer_agenda: pyarts.arts.Agenda | None = self.propagation_path_observer_agenda, frequency_grid: pyarts.arts.AscendingGrid | None = self.frequency_grid, zenith_grid: pyarts.arts.AscendingGrid | None = None, azimuth_grid: pyarts.arts.AscendingGrid | None = None) → None

Computes the spectral radiance field using propagation_path_observer_agenda.

Each point is in computed individually, so there will be zenith x azimuth x altitude x latitude x longitude x frequency number of calculations. The positional arguments are taken from spectral_radiance_operator.

If the code is not already in parallel operation mode when this method is called, the first 5 dimensions are computed in parallel.

Author(s): Richard Larsson

Parameters:

  • spectral_radiance_field (StokvecGriddedField6) – The spectral radiance field. [OUT]

  • spectral_radiance_operator (SpectralRadianceOperator, optional) – The spectral radiance operator. See spectral_radiance_operator, defaults to self.spectral_radiance_operator [IN]

  • propagation_path_observer_agenda (Agenda, optional) – Get the propagation path as it is obeserved. See propagation_path_observer_agenda, defaults to self.propagation_path_observer_agenda [IN]

  • frequency_grid (AscendingGrid, optional) – The discrete frequency grid. See frequency_grid, defaults to self.frequency_grid [IN]

  • zenith_grid (AscendingGrid) – The zenith grid. [IN]

  • azimuth_grid (AscendingGrid) – The azimuth grid. [IN]
