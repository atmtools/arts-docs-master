pyarts.arts.predef.get_self_h2o_ckdmt350

predef.get_self_h2o_ckdmt350(f_grid: pyarts.arts.Vector, rtp_pressure: float, rtp_temperature: float, x_h2o: float) → pyarts.arts.Vector

Computes self absorption using MT CKD version 3.50

Parameters:

  • f_grid (Vector) – Frequency grid [Hz]

  • rtp_pressure (float) – Pressure value [Pa]

  • rtp_temperature (float) – Temperature value [K]

  • x_h2o (float) – Ratio of water in the atmosphere in the range [0, 1]

Returns:

abs_coef – Absorption coefficients

Return type:

Vector
